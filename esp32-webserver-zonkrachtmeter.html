‚ö†Ô∏è Important Wiring Notes

  • Power: Use 3.3V, NOT 5V! The BH1750 is a 3.3V device
  • I2C Pins: GPIO 21 (SDA) and GPIO 22 (SCL) are the default I2C pins
  • Pull-up Resistors: Most BH1750 modules have built-in pull-up resistors
  • Multiple Sensors: You can connect multiple BH1750 sensors by changing their I2C address

8. üîß Troubleshooting

ESP32 won't connect to WiFi

Solutions: Check SSID/password spelling, ensure 2.4GHz network, verify signal strength, try different WiFi network

Sensor not detected / readings show 0 or -1

Solutions: Check wiring connections, verify 3.3V power, ensure I2C pins are correct (GPIO 21/22), try different sensor

Web page won't load

Solutions: Verify IP address from Serial Monitor, ensure phone is on same WiFi network, try refreshing page, check firewall settings

Readings seem inaccurate

Solutions: Clean sensor lens, ensure sensor isn't obstructed, calibrate against known light source, check for electromagnetic interference

System keeps rebooting

Solutions: Check power supply capacity, verify wiring doesn't cause shorts, add capacitor for power filtering, check for code infinite loops

10. üîß Maintenance & Tips

Regular Maintenance

  • Monthly: Clean sensor lens with soft, dry cloth
  • Quarterly: Check all connections for corrosion (outdoor installations)
  • Bi-annually: Update firmware if new features are available
  • As needed: Replace enclosure seals for weatherproofing

Optimization Tips

  • Placement: Position sensor to avoid direct shadows from nearby objects
  • Calibration: Compare readings with professional light meter for accuracy
  • Power: Use quality USB power supply (2A minimum) for stability
  • Network: Ensure strong WiFi signal at installation location
  • Backup: Keep spare ESP32 and sensor for quick replacement

Advanced Features You Can Add

  • Data logging to SD card or cloud storage
  • Email/SMS alerts for extreme light conditions
  • Multiple sensor support for different locations
  • Integration with home automation systems
  • Solar panel efficiency monitoring
  • Plant growth optimization alerts

Safety Considerations

  • Never expose electronics to water without proper enclosure
  • Use appropriate voltage levels (3.3V for sensor)
  • Ensure proper ventilation in enclosures to prevent overheating
  • Use surge protectors for outdoor installations
  • Follow local electrical codes for permanent installations
